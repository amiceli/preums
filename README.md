# Prums

Preums *(first in French)* is a side project aimed at reminding people that coding is fun without AI.

Remember first repo, first commit, first MR <3.

## Goals

A brief look back in time with first repositories, first contributor, first issues etc with Github Api.

## Technologies

**Laravel** (without Laravel Boost) and **Vue**, the frameworks that always make me want to code and continue in this profession. And
[NeoBrutalismCSS](https://github.com/matifandy8/NeoBrutalismCSS]) for a little retro flair.

And tools that are magical : biome, lefthook, just

## Run project

With just : 
- `just install`
- `just up` (with tmux)
- `just run`
